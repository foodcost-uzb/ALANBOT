# Инструкция для родителя

## Начало работы

1. Откройте бота в Telegram: **@childcontroll_bot**
2. Нажмите **Start** или отправьте `/start`
3. Выберите **"Я родитель"**
4. Введите пароль (по умолчанию: `1234`)
5. Бот выдаст вам **инвайт-код** — отправьте его ребёнку

## Команды

| Команда | Что делает |
|---------|-----------|
| `/today` | Прогресс ребёнка за сегодня: какие задачи выполнены, сколько баллов |
| `/report` | Отчёт за текущую неделю: баллы по дням + процент карманных денег |
| `/history` | Отчёты за последние 4 недели |
| `/extra` | Назначить ребёнку дополнительное задание с бонусными баллами |
| `/invite` | Показать инвайт-код (если нужно привязать ещё ребёнка) |
| `/password` | Сменить пароль для регистрации родителей |

## Как назначить доп. задание (`/extra`)

1. Отправьте `/extra`
2. Если детей несколько — выберите ребёнка из списка
3. Введите название задания (например: "Помыть посуду")
4. Введите количество бонусных баллов (или просто Enter для 1 балла)
5. Ребёнок получит уведомление о новом задании

## Что происходит автоматически

- **7:00** — ребёнку приходит чеклист на день
- **12:00 и 17:00** — ребёнку приходят напоминания (если есть невыполненные задачи)
- **21:00** — вам приходит итог дня по каждому ребёнку
- **Воскресенье 20:00** — вам приходит полный отчёт за неделю

## Уведомления от ребёнка

Когда ребёнок выполняет задачу и отправляет фото-подтверждение, вам приходит это фото с подписью, какая задача выполнена.

## Система баллов

Каждый день ребёнок может заработать **до 8 баллов** (по 1 за задачу):

**Утренние задачи:**
- Почистить зубы
- Одеться
- Заправить кровать

**Вечерние задачи:**
- Принять душ
- Сменить бельё
- Положить вещи в стирку
- Подготовить одежду на завтра
- Убрать за собой

**Важные правила:**
- Если ребёнок **не принял душ** — весь день считается как **0 баллов**
- В **воскресенье** добавляется задача "Уборка комнаты". Если не выполнена — **штраф -5 баллов** от недельного итога

## Карманные деньги

Процент от карманных денег зависит от суммы баллов за неделю (максимум 56):

| Баллы за неделю | Процент |
|-----------------|---------|
| 50–56 | 100% |
| 42–49 | 70% |
| 35–41 | 40% |
| менее 35 | 0% |

Дополнительные задания дают бонусные баллы сверх основных — они помогут ребёнку добрать до нужного уровня.
